## <p style="text-align: center; color: #383e42;"> ¡DESCUBRE TU PERSONALIDAD!</p>

**✏️ ¡Realiza este quiz interactivo y descubre cual es tu tipo de personalidad!**

---

### <p style="color: #383e42;"> ⚙️ ¿Cómo funciona? </p>

Cada respuesta suma puntos a una personalidad específica.  
Una vez realizado el cuestionario:

- Se cuentan los puntos obtenidos.
- Se identifica la personalidad con mayor puntaje.
- Se muestra el resultado correspondiente.

---

### <p style="color: #383e42;"> ▶️ ¿Cómo se ejecuta el proyecto? </p>

Para que usted pueda completar este quiz, siga los pasos a continuación:

1. Descargue o clone el repositorio.
2. Abra el archivo **index.html.**
3. Responda las preguntas.
4. ¡Descubra su personalidad!

---
